CTK 301 - Web Design and Development

Taken at Illinois State University, this class taught me the fundementals of good Web Design & Development

Learning HTML & CSS Fundementals

In this class, Assignments 1-4 were focused on learning the fundementals of CSS and HTML using a sample webpage.Using HTML we spent a lot of time learning things like lists, dividing tags (body, span, div), text tags (p,b,br) and how pages link together. Although quite basic in functionality and code, there was a lot of learning to be done here and quite a bit of experimentation and fun to be had.
After learning the basics of HTML we moved on to CSS and explored the vast land of Style Sheets. This included how to link them to pages, how to change fonts, colors, and even backgrounds. More basics, but you have to start somewhere right?

Recreating a Movie Poster

Assignment 5 put what we had learned throughout the class to use by having us recreate a Movie Poster. Although seemingly simple to many advanced Web Developers, to a beginner like me this was a time consuming task. There were various font colors and styles that had to be taken into consideration, and I wanted to get the spacing as close as possible to the example poster.
